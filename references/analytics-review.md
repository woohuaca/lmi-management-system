# Analytics Review

Use this reference for time design, personal productivity summary, statistical analysis, variance review, and management analysis.

## Time Image

Use a weekly time image to design the ideal rhythm before measuring actual usage.

- Place high-return activities first.
- Reserve recurring blocks for planning, coaching, recruiting, strategy, or other recurring value-creating work.
- Protect self-development time instead of leaving it to leftovers.
- Compare the ideal weekly image with the actual weekly summary to find drift.

## Analysis Levels

- `Result`: what happened
- `Variance`: where actual result diverged from target
- `Input`: where time, energy, and management attention actually went
- `Mechanism`: whether the management routines supported execution
- `Adjustment`: what should change next

## Preferred Outputs

- weekly time image
- work list for analysis
- daily hours by activity
- weekly total
- weekly average
- percentage of total time
- progress summary
- target variance
- high-return activity effectiveness
- time investment pattern
- root cause analysis
- recommended adjustments

## Root Cause Categories

- unclear or unrealistic target
- weak prioritization
- insufficient time protection
- low follow-through quality
- poor delegation
- missing management mechanism
- external dependency or resource constraint
