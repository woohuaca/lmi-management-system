# Weekly Planning And Review

This is the bridge between monthly goals and daily execution.

## Weekly Plan Rules

- Start from monthly priorities, not from calendar leftovers.
- Use the week to translate monthly goals into a small number of weekly wins.
- Make the weekly plan explicit enough that it can be reviewed later in a plan-versus-actual format.
- Link each major weekly goal to a role when the user manages through a role clarification table.
- Keep a visible area for this week's top goals and high-return activities.
- Distinguish `A` important items from `B` urgent items.
- Reserve `C` for contact or follow-up items and `D` for meetings, discussions, or coordination.
- Separate work the user must personally drive from work that can be delegated or followed up.
- Protect time for high-return actions before filling meetings and admin work.
- Include management rhythm actions for managers such as one-on-ones, checkpoints, team reviews, recruiting, or coaching.
- Add one or two review hooks so the end-of-week review can test whether the week succeeded.

## Weekly Review Rules

- Review outcomes, not just busyness.
- Start with a plan-versus-actual comparison, not a loose summary.
- Compare time spent against the stated weekly priorities and the intended weekly rhythm.
- Identify what created leverage and what created drag.
- Review the week through role lenses when the user works from a role clarification table.
- Separate three causes when results miss:
  - target issue
  - execution issue
  - mechanism issue
- Allow strategic adjustment when the user changed direction on purpose instead of simply missing execution.
- Extract lessons, validated methods, and repeatable workflows, not only problems.
- Carry only the most important unfinished work forward.
- Explicitly delete low-value unfinished work instead of rolling everything over.
- Convert the review into next week's protected blocks, delegated items, and key focus changes.
- End the weekly review with a light next-week GPS preview when the user is already thinking about the next cycle.

## Weekly Sections

- plan versus actual
- week theme
- linked monthly priority
- weekly top goals
- role review
- high-return activities
- urgent items
- important items
- delegated or follow-up items
- management rhythm
- protected time blocks
- lessons and insights
- review and carry-forward decisions
- next-week GPS preview

## Typical Weekly Questions

- What must be true by the end of this week?
- Which action creates the most leverage if completed?
- What did I spend time on that did not deserve the time?
- Which unfinished item should be deleted instead of rolled over?
- What mechanism should be tightened next week?
- Which role advanced most this week, and which role was under-served?
- What method from this week is worth reusing next week?
