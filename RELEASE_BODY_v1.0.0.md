# v1.0.0

`lmi-management-system` is the first complete public release of the skill.

## What This Release Includes

- role goal clarification
- monthly planning
- weekly planning
- daily work planning
- daily review
- weekly review
- monthly review
- role review
- time image design
- personal productivity summary and analysis

## Key Design Decisions

- separate planning from review
- classify daily work as `A/B/C/D`
- require unfinished work to be re-decided instead of endlessly rolling forward
- make role goals flow into month, week, and day
- separate reflections, ideas, and validated project facts

## Operational Additions

- `azai` usage guide
- OpenClaw execution playbook
- storage system guidance
- sync check script
- GitHub-friendly README
- MIT license

## Recommended First Prompt

```text
请使用 lmi-management-system 协同我，先判断我是月计划、周计划、日计划、日复盘、周复盘、月复盘、角色复盘、时间图像还是统计分析场景，然后输出一版草案。
```
